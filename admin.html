Accès admin

Réservé à l'organisation. Connexion avec le compte créé dans Supabase (Authentication → Users) — ce n'est plus un simple mot de passe partagé dans le code : les données des participant·es (emails, téléphones) ne sont désormais lisibles qu'une fois connecté·e.

Réglages du site

Décoché : le lien « Parrains » disparaît du menu et la page affiche un message « ça reste une surprise ! » à la place des vidéos — pratique tant qu'aucun parrain n'est encore annoncé. Coché : tout redevient visible normalement, sans avoir besoin de redéployer le site. Le changement est pris en compte sur le site public en quelques secondes (le temps de recharger une page).
